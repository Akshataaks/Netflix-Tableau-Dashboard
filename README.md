# Netflix Content Analysis Dashboard

This project analyzes Netflix content using Tableau.

## Tools Used

* Tableau
* Excel / CSV dataset

## Dashboard Features

* Total Titles KPI
* Movies vs TV Shows distribution
* Content by Country Map
* Top Genres Analysis
* Content Added by Year Trend

## Dataset

The dataset contains information about Netflix movies and TV shows including title, country, genre, release year, and rating.

## Project Files

* Tableau Dashboard (.twbx)
* Dataset (.csv)
* Project Report (.pdf)
